# PRD GAMES Web

Dokumen PRD utama berada di `prd_games_mvp_website.md`.

Ringkas scope MVP:

- Website publik informasi GAMES.
- Pendaftaran peserta dengan nomor registrasi otomatis.
- Cek status pendaftaran.
- Admin login dan CMS custom.
- Dashboard peserta, validasi pembayaran/berkas manual, export CSV.
- Supabase RLS untuk melindungi data peserta.

Fitur non-MVP seperti payment gateway, WhatsApp otomatis, sertifikat otomatis, ujian online, scoring realtime, dan dashboard juri tidak dikerjakan pada MVP.
